# Excel Dengue Dashboard

![Excel](https://img.shields.io/badge/Tool-Microsoft%20Excel-217346?logo=microsoft-excel&logoColor=white) ![Dashboard](https://img.shields.io/badge/Type-Health%20Dashboard-B8860B) ![Focus](https://img.shields.io/badge/Domain-Dengue%20Analysis-B22222)

## Overview
This project presents a dengue analysis dashboard built in Microsoft Excel. It combines cleaned data, geographic mapping inputs, and dashboard visuals to communicate outbreak patterns in a clear and decision-friendly format.

## Project contents
- `Dengue.xlsx`: main workbook with analysis and dashboard views.
- `Dhaka_Dengue_Map_Data.xlsx`: map-focused dataset used for geographic visuals.
- `Dengue project.png`: dashboard preview image.

## What this project highlights
- Data cleaning and transformation in Excel.
- KPI-oriented dashboard storytelling.
- Geographic perspective on dengue case distribution.
- A portfolio-ready example of spreadsheet analytics.

## How to use
1. Open `Dengue.xlsx` in Microsoft Excel.
2. Review source sheets and dashboard sheets.
3. Open `Dhaka_Dengue_Map_Data.xlsx` if you want to inspect or update map data.
4. Use `Dengue project.png` as a quick visual preview.

## Recommended tools
- Microsoft Excel (Office 2019 or Microsoft 365 preferred)
